Booking Grooming Appointments

Operating Procedure

 

Note: This procedure is for standalone grooming appointments only. Grooming services that are attached to a pre-existing reservation must be added on to the actual reservation itself.

 

  1. Navigate to the 'New Appointment' screen (Navigate to the left-hand-side menu, 'Reservations' > 'New Appointment'.)
  2. Add pet(s)'s name under the 'Select Pets' section. Verify last name is correct.
  3. Under 'Service Category', select the only option available, 'Grooming'
  4. Under 'Appointment Type', select desired service.
  5. Under 'Specialist', select:
  • For baths or nailtrims: always select 'Paws Bather'.
  • For full service groom, select the appropriate full service groomer.
