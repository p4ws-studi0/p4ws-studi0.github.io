Long Term Stays

Operating Procedure

 

  • If a pet stays for more than 2 weeks, we charge their card on file every week and apply a pre-payment to their reservation. We must let the client know of this, and we must keep a card on file for these situations.
  • Please take down primary, secondary, and emergency contacts for these situations.

 

Applying a pre-payment to a long-term reservation:

  • Navigate to the desired reservation.
  • Press the dollar sign icon on the top right of the screen to view the invoice.
  • Press the green 'Apply Payment' button. 
  • Calculate the amount the customer owes up to this point: find the nightly lodging rate times the number of nights they have stayed thus far. Then add the activity package and pet care warranty, times the number of days they have stayed thus far. Add any other items the client should have paid for thus far. Then add tax (multiply this number by 1.0635. (CT sales tax)). This is the total amount you should charge for the pre-payment.
    • Note: Make sure if there were any previous pre-payments, you are considering this when calculating the current pre-payment amount.
  • Charge the calculated amount using the card on file.
  • Note the file saying you have applied the pre-payment to the reservation, with the amount charged.
