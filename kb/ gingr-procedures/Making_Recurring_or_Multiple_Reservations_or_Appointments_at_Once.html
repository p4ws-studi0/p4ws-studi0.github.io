Making Recurring or Multiple Reservations or Appointments at Once

Operating Procedure

 

This procedure will outline how to add recurring or multiple reservations or appointments at once. This is an in-depth overview that also includes how to comprehensively set up a reservation or appointment. To jump straight to the recurring or multiple reservation part, see the information below highlighted in blue.

 

RESERVATIONS:

 

Recurring Reservations (Only for Daycare):

  • Navigate to the 'New Reservation' screen (Navigate to the left-hand-side menu, 'Reservations' > 'New Reservation'.)
  • Ensure pet(s)' name is selected under the 'Pet's' section. Ensure last name is correct and the correct pet(s) are selected.
  • Select correct booking type under the 'Type' section.
  • Select the first desired date.
  • To add recurring dates: 
  1. Press ‘Add Recurring Dates’ Button
  2. Select desired day(s) of week.
  3. If desired, select the frequency. NOTE: this will be rare you have to use this section.
  4. Select what date the recurring reservations stop.
  5. Press ‘Add Dates’.

  • Once all details are entered, press 'Save' at the bottom of the screen.
  • It will display a notice saying, "A lodging must be selected for each date of this reservation." Proceed to press 'Override & Create,' as we do not reserve rooms for daycare reservations in advance.

Making multiple reservations at once (Daycare or Boarding)

  • Navigate to the 'New Reservation' screen (Navigate to the left-hand-side menu, 'Reservations' > 'New Reservation'.)
  • Ensure pet(s)' name is selected under the 'Pet's' section. Ensure last name is correct and the correct pet(s) are selected.
  • Select correct booking type under the 'Type' section.
  • Select the first desired date(s).
  • Press the 'Add Another Reservation' button and enter date(s).
  • Add on any desired services and/or activities.
    • Note: if there are different services on each reservation, make them separately to ensure each reservation is correct.
  • Proceed to finalize reservation
    • Boarding: assign room, write any notes, and take the $50 deposits (one per reservation).
    • Daycare: do not assign room as usual, and write any notes
  • Once all details are entered, press 'Save' at the bottom of the screen.

  3. Under 'Service Category', select the only option available, 'Grooming'
  4. Under 'Appointment Type', select desired service.
  5. Under 'Specialist', select:
  • For baths or nailtrims: always select 'Paws Bather'.
  • For full service groom, select the appropriate full service groomer.

  • For nail trims, after the date is selected, select only the drop off time and skip to step 8.
