Creating an Incident Report

 

This procedure is designed to ensure that comprehensive notes are recorded and communicated in all situations / occurrences where a dog is observed to display any "abnormal" or concerning behavior or condition while in the care of Paws Pet Resort.

 

Pet Care staff are trained to know what to look for and what constitutes "abnormal" behavior. WHEN IN DOUBT, CREATE A NOTE IN THE PET'S FILE AND LET A MANAGER OR SUPERVISOR KNOW ABOUT THE BEHAVIOR. 

 

If the issue is serious and/or if there is an injury involved, create a new Incident Report for an animal: 

 

  1. Navigate to the pet's profile on Gingr. Then navigate to the Incidents section on the animal's profile. 
  2. Complete all required fields on the new Incident Report on Gingr. The more information the better! 
  3. Save
  4. Write notes in the file and tell management!
