Reception Tasks Before Opening

Operating Procedure

 

This is an overview of the important tasks reception must do before opening our doors.

 

Printing Boarding Arrival Room Cards

  1. Navigate to 'Reports & More' > 'Reports' > 'Print Run Cards'
  2. Select today's date.
  3. Under 'Reservation Type', select only the boarding options.
  4. Press 'Print Run Cards!' - DO NOT ACTUALLY PRINT THEM YET!
  5. At the top right of the screen, it will say 'Run Card 1 of [XX]' Look at the total number of run cards.
  6. In a new tab (not closing out of the old tab), access the 'Run Card' document on the 'Paws HQ' Google Sheets document.
  7. Print out the correct total number of run card bottoms. 
  8. Once the correct number of run card bottoms are printed, put them in the printer tray faced down, with the printed material at the top.

 

 

  1. Navigate back to the old tab with all the arrival room cards, and press print.
  2. The room cards have been successfully printed.
  3. On the top left of each room card for dogs only, write what type of play they go to (individual, big, small, or medium). If the dog needs to be evaluated for group, write 'eval'. 
  4. For all areas besides Luxury Suites and Cat Rooms, fold paper in half and put it in a plastic sleeve.
  5. Place all room cards on the table next to the R2 desk in alphabetical order and separated by area for easy access. Also, make sure there is tape to hang up the Luxury Suite and Mini Mansion room cards.

 

Printing ID collars for all arriving boarding dogs

  1. Note: only dogs get ID collars.
  2. On the 'Expected Today' section on the dashboard, search 'dog boarding'. Filter by lodging type to sort pets by area.
  3. Open every filtered reservation in a new tab - to do this, click on the text for the type of the reservation (ex: 'Standard Dog Boarding'). Either right click the text and press open in a new tab, or simply push the scroll wheel down to open the reservation in a new tab. Proceed to open all reservations in their own new tab.
  4. Navigate to the first tab you opened, and press the print button at the top right, and then click 'Print Labels'.
  5. This will give you access to a printing preview screen. (Note: if this does not show up, it is because pop-ups are disabled. You must enable popups in order for this screen to show up.)
  6. Press CTRL + P to print - ensure the destination is set to 'Petdetect Raster Printer'
  7. Under 'Copies' you must use the chart below to determine how long the collar will be.
Smaller Breeds (<35 Lbs) Medium or Larger Breeds (~35-80 Lbs) (Most Common) Extra Large Breeds (>80 Lbs)
3 Labels Long 4 Labels Long 5 Labels Long

 

  1. Once the correct number of copies is entered and the correct printing destination is set, print out the label.
  2. Labels should be printing now - if they are not,
  • We have a switch (on the cash drawer) to swap between printing from the R1/R2 computer. Ensure that the light on the switch refects the computer you are currently using.
  • Ensure there is a roll of label tape fed into the label printer (the colored side should be down - the roll of label tape fed into the machine should be the one on the left. There must be three rolls of label tape, side-by-side, in order for the label printer to work properly. Again - only the leftmost roll should be fed into the machine.
  • If the label machine is stalling/not printing well, ensure the roll is aligned perfectly, with no additional slack.
  • If the label machine is not cutting each collar well, wipe off the blade CAREFULLY with a disinfectant wipe. Remove all built-on gunk with the wipe and it should cut perfectly after.
  1. Proceed to print all ID collars.
  2. Once all ID collars are printed, stick them to the back wall behind the reception desk in room order (remove one smaller section of the back of the sticker and place on wall).
  3. Double-check that all ID collars have been correctly printed.
